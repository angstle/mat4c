3  Übungsaufgaben Quadratische Gleichungen

3.1 Gemischte Aufgaben

  1. Geben Sie jeweils die Lösungsmenge der quadrierten Gleichung an.
  1. \((x - 3)^2 = 16\)
  2. \((x - 4)^2 = 81\)
  3. \((2x - 3)^2 = 16\)
  4. \((x + 5)^2 = (x - 3)^2\)
  5. \((2x - 1)^2 = (x + 3)^2\)
  6. \((2x - 3)^2 = (3x + 8)^2\)
  7. \((2x - 5)^2 = (x + 4)^2\)
  8. \((2x - 3)^2 = (2x + 4)^2\)
  9. \((2x - 3)^2 - (x - 6)^2 = 0\)
Lösung
  1. \(\mathbb{L} = \{7,\,-1\}\)
  2. \(\mathbb{L} = \{13,\,-5\}\)
  3. \(\mathbb{L} = \{\tfrac{7}{2},\,-\tfrac{1}{2}\}\)
  4. \(\mathbb{L} = \{-1\}\)
  5. \(\mathbb{L} = \{4,\,-\tfrac{2}{3}\}\)
  6. \(\mathbb{L} = \{-11,\,-1\}\)
  7. \(\mathbb{L} = \{9,\;\tfrac{1}{3}\}\)
  8. \(\mathbb{L} = \{-\tfrac{1}{4}\}\)
  9. \(\mathbb{L} = \{-3,\;3\}\)
  1. Wählen Sie jeweils eine geeignete Lösungsmethode und geben Sie die Lösungsmenge der quadratischen Gleichung an.
  1. \((x - 3)(x + 4) = 0\)
  2. \(x^2 - 144 = 0\)
  3. \(x^2 - 5x + 6 = 0\)
  4. \((2x + 6)^2 = 49\)
  5. \(x^2 - x = 20\)
  6. \(5x^2 = 125\)
  7. \(x^2 - 2x - 63 = 0\)
  8. \((2x + 7)^2 = 289\)
  9. \(3x^2 - 300 = 0\)
Lösung
  1. \(\mathbb{L} = \{3,\,-4\}\)
  2. \(\mathbb{L} = \{12,\,-12\}\)
  3. \(\mathbb{L} = \{2,\;3\}\)
  4. \(\mathbb{L} = \{\tfrac{1}{2},\,-\tfrac{13}{2}\}\)
  5. \(\mathbb{L} = \{5,\,-4\}\)
  6. \(\mathbb{L} = \{5,\,-5\}\)
  7. \(\mathbb{L} = \{9,\,-7\}\)
  8. \(\mathbb{L} = \{5,\,-12\}\)
  9. \(\mathbb{L} = \{10,\,-10\}\)
  1. Wählen Sie jeweils eine geeignete Lösungsmethode und geben Sie die Lösungsmenge der quadratischen Gleichung an.
  1. \(x^2 - 7x = 120\)
  2. \(-2x^2 = 98\)
  3. \((3x + 12)(x - 11) = 0\)
  4. \((2x + 6)^2 = 49\)
  5. \((x + 6)^2 = (x + 7)^2\)
  6. \(x^2 - 5x - 14 = 0\)
  7. \((2x + 3)^2 - 169 = 0\)
  8. \(x^2 + 16 = 0\)
  9. \(x^2 = (5x - 18)^2\)
Lösung
  1. \(\mathbb{L} = \{15,\,-8\}\)
  2. Keine reelle Lösung (\(\mathbb{L} = \{\}\))
  3. \(\mathbb{L} = \{-4,\;11\}\)
  4. \(\mathbb{L} = \{\tfrac{1}{2},\,-\tfrac{13}{2}\}\)
  5. \(\mathbb{L} = \{-\tfrac{13}{2}\}\)
  6. \(\mathbb{L} = \{7,\,-2\}\)
  7. \(\mathbb{L} = \{5,\,-8\}\)
  8. Keine reelle Lösung (\(\mathbb{L} = \{\}\))
  9. \(\mathbb{L} = \{\tfrac{9}{2},\;3\}\)
  1. Wählen Sie jeweils eine geeignete Lösungsmethode und geben Sie die Lösungsmenge der quadratischen Gleichung an.
  1. \((x + 3)^2 = 0\)
  2. \((8 - x)^2 = 144\)
  3. \(x^2 - 5x = 24\)
  4. \((3x - 16)^2 = (4x - 19)^2\)
  5. \((11x + 12 - 3x)(11 + 12x - 3) = 0\)
  6. \((7x - 5)^2 = (2x + 5)^2\)
  7. \(x^2 - 15x = -50\)
  8. \((4 - 3x)^2 - (2x + 5)^2 = 0\)
  9. \((4x - 9)^2 = 36\)
Lösung

  1. \(\mathbb{L} = \{-3\}\)

  2. \(\mathbb{L} = \{-4,\;20\}\)

  3. \(\mathbb{L} = \{8,\,-3\}\)

  4. \(\mathbb{L} = \{3,\;5\}\)

  5. Gleichung vereinfacht zu \((8x+12)(12x+8)=0\), Lösungen: \(\{-\tfrac{3}{2},\,-\tfrac{2}{3}\}\)

  6. \(\mathbb{L} = \{2,\;0\}\)

  7. \(\mathbb{L} = \{10,\;5\}\)

  8. \(\mathbb{L} = \{-\tfrac{1}{5},\;9\}\)

  9. \(\mathbb{L} = \{\tfrac{15}{4},\;\tfrac{3}{4}\}\)
