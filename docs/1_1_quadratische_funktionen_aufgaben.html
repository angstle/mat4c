Formen der quadratischen Funktion

Allgemeine quadratische Funktion (Scheitelpunktform)

  1. Bei der rein quadratischen Funktion haben wir gesehen, dass diese immer eine Nullstelle bei \(x = 0\) hat. Wir möchten uns nun anschauen wie viele Nullstellen allgemeine quadratische Funktionen haben.

    1. Überlegen Sie sich, wie viele Nullstellen quadratische Funktionen haben (verschiedene Fälle).
    2. Skizzieren Sie ein Beispiel für eine quadratische (nicht rein quadratische) Funktion mit genau einer Nullstelle. Finden Sie auch eine mögliche Funktionsgleichung?
    3. Skizzieren Sie ein Beispiel für eine quadratische Funktion ohne Nullstelle. Finden Sie auch eine mögliche Funktionsgleichung?
    4. Skizzieren Sie ein Beispiel für eine quadratische Funktion mit zwei Nullstellen. Finden Sie auch eine mögliche Funktionsgleichung?
    5. Skizzieren Sie ein Beispiel für eine quadratische Funktion mit unendlich vielen Nullstellen. Finden Sie auch eine mögliche Funktionsgleichung?
    6. Finden Sie Bedingungen für die Parameter \(a, u, v\) so, dass eine quadratische Funktion keine, eine oder zwei Nullstellen hat.
Lösung
  1. Quadratische Funktionen können keine, genau eine oder 2 Nullstellen haben
  2. Beispiel: \(f(x) = (x-1)^2\)
  3. Beispiel: \(f(x) = (x+1)^2 + 4\)
  4. Beispiel: \(f(x) = (x-3)^2 - 4\)
  5. Unendlich viele Nullstellen sind nur bei der Nullfunktion \(f(x) = 0\) möglich
    1. Zwei Nullstellen: \(a \ne 0\), \(v < 0\)
    2. Eine Nullstelle: \(a \ne 0\), \(v = 0\)
    3. Keine Nullstelle: \(a \ne 0\), \(v > 0\)
  1. Eine Parabel hat den Scheitel \(S(2|3)\) und verläuft durch den Punkt \(P(4|−5)\). Bestimmen Sie die Funktionsgleichung der Parabel.
Lösung

Optimierungsprobleme

  1. Zahlenrätsel

    1. Welche beiden Zahlen, deren Summe \(12\) beträgt, haben das grösste Produkt?
    2. Welche beiden Zahlen, deren Diﬀerenz \(2\) beträgt haben das kleinste Produkt.
    3. Für welche Zahl wird das Produkt aus dem Dreifachen und der um \(1\) vergrösserten Zahl am kleinsten.
Lösung
  • \(x+y = 12 \rightarrow y=12-x\)
  • Gesucht: Maximiere \(f(x)=x\cdot (12-x) = -x^2 + 12x\)
  • Scheitelpunkt: \(x=\frac{-b}{2a} = \frac{-12}{2\cdot (-1)} = 6\)
  • \(y= 12-6 = 6\)

Lösung: Die beiden gesuchten Zahlen sind \(6\) und \(6\).

  • \(x-y = 2 \rightarrow y=x-2\)
  • Gesucht: Das minumum der Funktion \(f(x)=x(x-2) = x^2-2x\)
  • Scheitelpunkt: \(x=\frac{-b}{2a} = \frac{2}{2} = 1\)
  • \(y = 1-2 = -1\)

Lösung: Die beiden gesuchten Zahlen sind \(-1\) und \(1\)

  • Die gesuchte Zahl: x
  • Gesucht: das Minumum von \(f(x) = 3x(x+1) = 3x^2 + 3x\)
  • Scheitelpunkt: \(x=\frac{-b}{2a} = \frac{-1}{2\cdot 3} = -\frac{1}{2}\)

Lösung: die gesuchte Zahl ist \(-\frac{1}{2}\)

  1. Judith will mit \(11m\) Maschendraht einen rechteckigen Platz für ihren Hund einzäunen.

    1. Sie benutzt die Wände von Garage und Haus. Für welche Abmessungen wird der Platz am grössten? Geben Sie den Flächeninhalt an.
    2. Wie hätte sie die Abmessungen wählen müssen, wenn sie eine \(3m\) lange Mauer einbeziehen würde.

Lösung
  • Länge linke seite: \(a\), Länge untere Seite: \(b\), \(a+b=11m\)
  • \(A=a\cdot b \rightarrow f(a) = a\cdot (11-a) = 11a-a^2 = -a^2 + 11a\)
  • Gesucht: Maximum der Funktion \(f(a)\), bzw. deren Scheitelpunkt
  • Scheitelpunkt: \(a=\frac{11}{-2} = 5.5\)
  • \(a=5.5m\), \(b=5.5m\)

Lösung: Der Zaun hat eine Länge von 5.5m und 5.5m

  • Zaunlänge \(11m = 2a+2b-3m\)
  • \(a+b=7 \rightarrow a = 7-a \rightarrow f(a) = -a^2 + 7a\)
  • Gesucht: Maximum, bzw. Scheitelpunkt von \(f(a) \rightarrow a=\frac{7}{-2} = 3.5, b=7-35. = 3.5\)

Lösung: Die maximale Fläche wird mit den Seitenlängen \(3.5m\) und \(3.5m\) erreicht.

  1. Ein quadratischer Tisch mit der Seitenlänge \(2m\) soll wie abgebildet mit zwei quadratischen Einlegearbeiten verziert werden. Aus Kostengründen soll dieser Flächenanteil möglichst klein werden. Welche Masse bieten sich für die Einlegearbeiten an?

Lösung
  • Tischfläche: \(A=4m^2\), sei \(a\) die Seitenlänge des kleineren und \(b\) die Seitenlänge des grösseren Einlegequadrats. Die gesamte Fläche der Einlegequadrate beträgt: \(A=a^2 + b^2\). \(a+b=2\)
  • Es gilt: \(b = 2-a\) und somit folgt durch einsetzen: \(A=a^2 + (2-a)^2 = 2a^2 -4a +4\)
  • Gesucht: Minimum der Funktion \(f(a)\rightarrow a=\frac{4}{4}=1\)
  • \(b = 2-a = 2-1 = 1\)

Lösung: Die minimale Fläche der Einlegequadrate wird mit den identischen Seitenlänge beider Quadrate von \(a=1m\) und \(b=1m\) erreicht.
